This is sample project for Server Driven UI using Kotlin Multiplatform for Android and iOS.

![Image](https://thinkuldeep.com/images/kmp-0.jpg)

This project build a simple end-to-end SDUI system using:
- Spring Boot (Server)
- Kotlin Multiplatform (Client)
- Compose Multiplatform (Rendering)
- Ktor Client (Networking)
